1)DOCUMENT Object:-
             The document object represent a web page that is loaded in the browser.
By accessing the element in the HTML page.With the help of document objects,we can 
add dynamic content to our web page.The documentobject can be accessed with a window.document
or just document.
 
PROPERTIES OF DOCUMENT Object:-
             Document objects such as title,body,cookies,etc .also be accessed by a window like this
window. document.title

SYNTAX:-
            document.propertyname;



2)WINDOW Object:-
             The window object is the topmost object of the DOM hierarchy.It represents a browser
window or frame that displays the contents of the webpage.Whenever a window appears on the screen 
to display the contents of the document,the window object is created.

PROPERTIES OF THE WINDOW Object:-
             Window object cannot be accessed by the document object.

SYNTAX:-
             window.propertyname;
